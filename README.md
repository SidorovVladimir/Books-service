### Books Service

1. Clone the application repository with the command:

```
git clone https://github.com/SidorovVladimir/Books-service.git
```

2. Go to the directory using the command:

```
cd Books-service
```

#### Installation

```
make setup
```

#### Running the app

```
make start
```
